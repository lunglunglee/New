Welcome to your MVC Web App built for reusability!

Your new MVC Web App is build on your reusable framework powered by the Genesys Framework.

Genesys Framework helps you create a reusable framework for your business entity objects and view-model objects ...allowing your framework objects to work in MVC, WebAPI and Cross-Platform Universal Apps. Works with your existing SQL tables, and allows you to incrementally build your reusable framework one object, one page at a time.

What do I do next?

Step 1: Publish the Framework.Database to your SQL Server by double-clicking Framework.Database\Publish\PublishToDev.publish.xml. Remember to set the SQL connection by clicking the Edit button.
Step 2: Set the Framework.Entity connection strings to your SQL Server in Framework.WebApp\Local\ConnectionStrings.config.
Step 3: Run the Framework.WebApp project to see the Customer Search pull a customer through CustomerInfo DAO and display the CustomerModel view model.

To learn more, visit http://GetFramework.com

This MVC Web App consists of:

  • Framework.WebApp - MVC Web App project. Small starter MVC web app that creates, reads, updates and deletes your reusable framework objects in this solution.
  • Framework.Database - SQL Server Data Tools project. Contains view/SP layer for the Framework.Entity to consume.
  • Framework.Entity - Entity Framework/Data Access Object project. Contains EF EDMX (database first), Data Access Objects, SaveableEntity/ReadOnlyEntity base classes.
  • Framework.Models - Cross-Platform View Models project. Contains the framework-level view models for http transport and .Serialize() built-in.
  • Framework.Interfaces - Cross-Platform Interface project. All projects reference this project so that interfaces are enforced in all tiers, and on mobile device as well.
